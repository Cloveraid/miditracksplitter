# miditracksplitter
Midi track splitter

* CHATGPT를 이용하여 만들어짐.

이 코드는 미디 파일을 분해해서 각 트랙별로 따로 저장하도록 만들어졌음.

저장된 트랙은 4/4 박자에 피치가 제거된 형태로 저장됨.
